# Scrambord
## How To Play
- Place down Letters from your Hand to create words on the Board.
- Submit Board using If valid, new Letters will be added to Hand.
- Redraw using then select any amount of Letters. Click so new Letters will be dealt.
- Expansion Points (ExP) can be used to reduce Strikes (Click Strikes Bar) or Expand (Arrows).
- There are Special Tiles. Reds being Penalties. Greens being Bonus Points. Oranges being Score Multiplier. Purples being Redraw. Blues being Random Expansion.
- Try to get as much points as possible whilst not striking out
